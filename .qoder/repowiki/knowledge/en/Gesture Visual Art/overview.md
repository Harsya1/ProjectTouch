The Gesture Visual Art project is a real-time interactive generative art application in Python. It captures webcam video, detects hand pinch gestures via MediaPipe, and spawns locked rectangular regions on the canvas that each display a unique generative visual effect (sketch portrait, glitch abstract, mosaic grid, point cloud depth scan). The visual reference is a TouchDesigner prototype captured as 300 PNG frames in `Frame Prototype/`. The application targets 20-30 FPS at 1280x720 resolution with up to 5 simultaneous regions.
